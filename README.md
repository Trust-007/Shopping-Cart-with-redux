# Shopping-Cart-with-redux

> This is a demo shopping app that let users add item to the cart.

## Built With

- React
- Redux
- firebase

## Live Demo

You can check out the project using this [live demo link](https://trust-cart-redux.netlify.app/)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- A web browser
- A code editor

### Setup

Assuming you have git installed on your computer, please follow the steps below:

- Create a folder
- Navigate to that folder in the command line using the `cd` command
- Clone the repository using the following command: `git clone https://github.com/Trust-007/Shopping-Cart-with-redux.git`
- `cd` into the project
- In the terminal run `npm install` and then `npm start`

you should see the live demo on 'localhost:3000'
